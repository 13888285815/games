# 4399游戏世界 - 类似4399.com的游戏网站

这是一个仿4399.com的游戏网站，提供各种在线HTML5小游戏。

## 🎮 功能特性

### 1. 网站功能
- **游戏分类浏览**：动作、益智、射击、赛车、休闲、体育、棋牌、策略等8个分类
- **热门游戏推荐**：基于游玩人数和评分的热门游戏推荐
- **游戏搜索**：快速搜索游戏名称、描述或分类
- **排行榜系统**：显示最受欢迎的游戏排名
- **轮播图展示**：展示热门和新上线的游戏
- **响应式设计**：适配桌面和移动设备

### 2. 游戏功能
- **贪吃蛇游戏**：经典贪吃蛇，支持键盘和触摸控制
- **俄罗斯方块**：经典俄罗斯方块，包含等级系统和分数计算
- **游戏进度保存**：本地保存游戏进度和最高分
- **用户偏好设置**：音效、图形质量等个性化设置

### 3. 技术特性
- **纯HTML/CSS/JavaScript实现**：无需额外依赖
- **Canvas游戏开发**：使用HTML5 Canvas实现游戏
- **响应式设计**：使用CSS Grid和Flexbox布局
- **游戏数据管理**：使用JavaScript对象管理游戏数据
- **本地存储**：使用localStorage保存用户数据和游戏进度

## 🚀 快速开始

### 方式一：直接打开（推荐）
1. 克隆或下载项目到本地
2. 使用浏览器直接打开 `public/index.html` 文件
3. 开始游玩游戏！

### 方式二：本地服务器
1. 安装Python 3
2. 在项目根目录运行：
   ```bash
   python -m http.server 8000
   ```
3. 访问 http://localhost:8000/public/

### 方式三：Node.js服务器
1. 安装Node.js
2. 运行：
   ```bash
   npx serve public
   ```
3. 按照提示访问网址

## 🔄 开发和扩展

### 添加新游戏
1. 在 `games/` 目录创建HTML游戏文件
